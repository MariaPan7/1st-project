HTML Table Reference

Table Tags

Tag Name Description
table Table The Wrapper element of all HTML tables
thead Table Head The set of rows defining the column headers in the table
tbody Table body The set of rows containing actual table data
tr Table Row The table row container
td Table data The table row container
tfoot Table Foot The set of rows defining the footer in the table

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td el should span
rowspan Row Span Defines how many rows a td el should span
